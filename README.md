# Java DSA Practice

This repository contains my Data Structures and Algorithms practice in Java.

The goal of this repository is to build strong problem-solving skills for coding interviews, placements, and improve my understanding of core DSA concepts.

## Topics Covered

* Basics of Java
* Math Problems
* Arrays
* Strings
* Recursion
* Sorting
* Searching
* Linked List
* Stack
* Queue
* Trees
* Graphs
* Dynamic Programming

## Structure

Each topic is organized into separate folders with problem-wise solutions.

Example:

* `Math/`
* `Arrays/`
* `Strings/`

* College/Placement Practice
